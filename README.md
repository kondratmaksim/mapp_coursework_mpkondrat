# mapp_coursework_mpkondrat

Название приложения: Dice Roller

Вариант курсовой работы: 5

Краткое описание назначения приложения:
Приложение «Dice Roller» — это симулятор бросков игральных костей для настольных игр. Оно позволяет выбирать тип и количество кубиков (от d4 до d20), автоматически вычисляет сумму результатов и сохраняет историю всех бросков в локальной базе данных с возможностью их быстрого повторения.

**Инструкция по запуску проекта:**
1. Синхронизация Gradle:
   ◦ После открытия проекта в Android Studio дождитесь появления уведомления о необходимости синхронизации и нажмите "Sync Now".
   ◦ Если уведомления нет, нажмите на иконку слона (Sync Project with Gradle Files) на панели инструментов.
2. Сборка проекта:
   ◦ Перейдите в меню Build -> Clean Project, чтобы удалить временные файлы старых сборок.
   ◦ Затем выполните Build -> Rebuild Project. Это сгенерирует необходимые классы для работы базы данных Room и навигации.
3. Запуск:
   ◦ Выберите подключенное устройство или эмулятор (рекомендуется API 30 и выше).
   ◦ Нажмите зеленую кнопку Run (Shift + F10).

**Основные реализованные функции:**

• Бросок кубиков: Выбор типа кубика (d4, d6, d8, d10, d12, d20) и их количества (от 1 до 10) через выпадающие списки (Spinner).
• Результаты: Отображение суммы броска и списка индивидуальных значений (если кубиков больше одного).
• История (Room): Все броски автоматически сохраняются в локальную базу данных с указанием даты и времени.
• Навигация: Переход на экран истории и обратно с помощью Jetpack Navigation Component.
• Восстановление настроек: Нажмите на любой элемент в списке истории, чтобы вернуться на главный экран и автоматически установить те же настройки (тип и количество кубиков), которые были в этом броске.
• Очистка: Возможность полностью удалить историю бросков нажатием на кнопку «Корзина» на экране истории.
